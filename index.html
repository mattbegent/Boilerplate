n. A rock solid front-end HTML5 and LESS framework for building optimised websites

LESS

Staddle includes base styles, CSS3 mixins and helpers to help you get started quickly.

JavaScript

Staddle uses RequireJs for loading modules and uses Grunt to lint and minify all JavaScript files into a single file.

Grunt Tools

Grunt is used to compile LESS, JSHint, minifying RequireJS, optimising images, LiveReload, starting a development server and generating static websites.

Static Site Generator

Staddle uses Assemble and Grunt to generate static websites. This compiled and optimised website can be put live on any server.

Responsive Grid

Staddles includes a simple to use and responsive grid, which includes breakpoints for tablet and mobile.

IE8+ support

IE8 support will be dropped when clients stop demanding it.

Quick Start

Get the latest version of Staddle:

bower install staddle

Or

git clone git@github.com:mattbegent/staddle.git

Or download the latest version using the download button at the top of the page.

Install node packages.

cd brilliantwebsitedirectory
npm install

Run Grunt tasks:

grunt

Open http://localhost:8080/ in your browser.

Build a cool website:-).
